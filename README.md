# BlueLogic Digital (OPC) Pvt. Ltd - Sky Blue Edition

This repository contains the final Sky Blue themed website for BlueLogic Digital (OPC) Pvt. Ltd.

## Files
- index.html - main website
- deploy.sh - helper script to push to GitHub
- favicon.ico / favicon.png - site favicon

## Deploy
Make deploy script executable and run it:
chmod +x deploy.sh
./deploy.sh
